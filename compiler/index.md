# Cherri Compiler Documentation

Welcome to the Cherri programming language compiler documentation!

If you are looking for documentation for the Cherri programming language itself, [click here](https://electrikmilk.github.io/cherri/language/).

Contributing to this open-source documentation is recommended as this language is in the early stages and while most
things work, bugs or inaccurate documentation are very likely.

**NOTE:** Keep in mind that this compiler is in the early stages of development. There is much refactoring to be done, the compiler has not yet reached the desired level of maturity and simplicity.
